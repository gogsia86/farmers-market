## ğŸ’¡ KEY LEARNINGS

### Technical Patterns Discovered

1. **Double Type Assertions for Mocks**

   ```typescript
   req as unknown as NextRequest;
   ```

   - Bypasses strict type overlap checking
   - Safe for test mocks implementing required interfaces

2. **Generic Constraints for Components**

   ```typescript
   <T extends Record<string, any> = Record<string, any>>
   ```

   - Ensures object-like structure for React props
   - Fixes IntrinsicAttributes errors

3. **PowerShell Bulk Refactoring**

   ```powershell
   (Get-Content file.ts -Raw) -replace 'pattern', 'replacement' | Set-Content file.ts
   ```

   - Extremely effective for mechanical changes
   - Reduced 9 error fixes from hours to seconds

4. **Prisma Type Patterns**

   ```typescript
   // Database-centric (Prisma) vs User-centric (UI)
   // Keep separate, convert at boundaries
   new Prisma.Decimal("19.99"); // For monetary values
   user.farmId || null; // Explicit null for optional fields
   ```
