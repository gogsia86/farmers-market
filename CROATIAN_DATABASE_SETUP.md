**Solution:**
```bash

# Or check .env.local
```

### Issue: Seed script fails
**Solution:**
```bash

npm run db:seed
```

### Issue: Croatian characters display incorrectly
**Solution:**
- Ensure PostgreSQL database uses UTF-8 encoding
- Check frontend uses `<meta charset="UTF-8">`
- Verify Next.js i18n configuration
